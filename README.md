# Telegram Video/Audio Downloader Bot

Ushbu bot YouTube va Instagram havolalaridan video va audio (MP3) yuklab olish uchun ishlatiladi.

## Texnologiyalar
- **Aiogram 3**
- **yt-dlp**
- **FFmpeg**
- **Docker**

## O'rnatish
Hugging Face'da **Settings -> Variables and Secrets** bo'limiga o'ting va quyidagi sirli kalitlarni (Secrets) qo'shing:
- `BOT_TOKEN`: Telegram bot tokeningiz
- `ADMIN_USERNAME`: Admin foydalanuvchi nomi (masalan, @ixlos_admin)
